Step-by-Step Setup Guide
1

Create a Google Account (skip if you have Gmail)

You need a Google account. If you already have Gmail, you're set. Otherwise create a free one at accounts.google.com/signup.

2

Go to Google Cloud Console

Open console.cloud.google.com and sign in. You'll land on the dashboard.

📍 You should see a top bar with a project dropdown and the Google Cloud logo.
If it asks you to agree to Terms of Service — check the box and click Agree and Continue
3

Create a New Project

At the top of the page, click the project dropdown (it may say Select a project or show an existing name). In the popup, click New Project in the top right.

• Project name: type anything — e.g. LeadFinder
• Leave Organization as-is
• Click Create
• Wait a few seconds, then make sure your new project is selected in the dropdown
4

Set Up Billing (Required by Google — you won't be charged)

Google requires a credit card to activate APIs, but their free tier covers all normal use. You will not be automatically charged.

• Click the ☰ menu (top left hamburger icon)
• Click Billing
• Click Link a billing account or Create billing account
• Enter your country → agree to terms → enter card
• Click Start Free

💡 You get $300 free credit for 90 days PLUS $200/month permanently for Maps APIs. Normal use stays free indefinitely.
5

Enable the Places API

• Click ☰ → APIs & Services → Library
• In the search box type: Places API
• Click the result called Places API (New)
• Click the blue Enable button
• A popup may appear showing your API key — copy it and paste it in the field above
6

Enable the Geocoding API

• Go back to APIs & Services → Library
• Search: Geocoding API
• Click it → click Enable
7

Get Your API Key

• Click ☰ → APIs & Services → Credentials
• Click + Create Credentials → API Key
• A box appears with your key starting with AIza…
• Click the copy icon next to the key
• Paste it into the field above and click Verify
